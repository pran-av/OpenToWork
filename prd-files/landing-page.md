# Landing Page

Unlike the Prelude Page and Pitch Page - the goal here is to make a very simple SaaS style website that is responsive for all mobile screen types, desktop and tablet.

## Design
Primary Poppins - use for Headings, Subheadings, Logos
Secondary Inter - use for CTA text, everything else

Color Theme: Sunrise Dawn. Warm. Trust. Excited. Optimistic. Use a pattern in the background as seems best to you. Make sure readability is 100% on all resolutions.

## Boiler Plate - P0

### Header: Name of the app "Pitch Like This". CTA "Create Your Pitch" on top right. For mobile have a shorter CTA name "Login". Fixed Header as user scrolls, for all platforms.
