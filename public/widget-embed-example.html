OpenToWork Widget Embed Example

Embed Code

Copy and paste this code into your website where you want the widget to appear:

<div id="my-action-button-widget" data-widget-id="WIDGET_ID_123"></div>
<script src="http://localhost:3000/widget-loader.js" async></script>

Note: Replace WIDGET_ID_123 with your actual widget ID, and update the script URL to your production domain.
